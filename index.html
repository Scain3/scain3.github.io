Sew Me

Sew Me is an e-commerce application that utilizes an Express backend and a React / Redux frontend. This application allows user to feel a part of the cusomization of clothing process. An authenticated user can purchase either a sewing pattern(a blue print for making the clothes), fabric, or a tailor made object. In order to request a tailor-made object the user must first provide their measurements, then the selected pattern, and lastly the selected fabric. Any user can search for a desired pattern or fabric but will be prompted to log in if attempting to add to cart or create a tailor made item. Please click on the Demo Login button to view the differences with an authenticated user view.

SIXPENCE

Sixpence is a clone of Etsy with a wedding essentials twist. This application is targeted toward non-sentimental people that want to sell their gently used items and people planning their wedding interested in buying these gently used items at a discounted price. The features used in the implementation of this app include: product listing, single product listing, create a product listing, add to cart, delete from cart, and search for a product listing.

Technologies

This application uses an Express backend and a React frontend. I utilize the Redux library to to ensure that the application state is available across the application.

Cheonjae

Cheonjae is Korean for Genius and is a clone of the music lyrics website called Genius. Cheonjae has a niche for only featuring Korean-Pop or K-POP music. Cheonjae's features include the ability for an authenticated user to add song lyrics and edit song annotations. All users can search and read lyrics provided on the website.

Technologies

This application integrates a Flask backend and a React / Redux frontend. This application was a Group Project where I worked mostly on the Search feature from backend to frontend and styling the navbar and home page. I leveraged Express to query the backend for pattern and fabric related information and then dynamically rendered customization options to users on the front end, based on their selections.
